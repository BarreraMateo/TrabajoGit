GIT BRANCH:

Este comando muestra todas las ramas locales, mientras que si le agregamos un nombre al final, crea una nueva rama con esa denominación. Además, si queremos eliminar una rama, agregando un -d y el nombre ya es eliminada. 



GIT MERGE: 

Este comando une todas las líneas de desarrollo disponibles en una sola (en la que estamos ubicados actualmente, esto lo podemos ver con un git checkout).
