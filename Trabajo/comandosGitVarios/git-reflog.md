# git reflog

## Como se usa?

Mostrar el reflog de una rama específica:

```bash
git reflog <nombre de la rama> 
```
Eliminar una entrada específica del registro:
```bash
git reflog delete <referencia>
```
## Para que sirve?

git reflog se utiliza para llevar un registro de seguridad en git para recuperar trabajo perdido

# git blame

```bash
git blame <archivo>
```

## Para que sirve?
Muestra quien modifico cada linea de un archivo y cuando lo hizo
